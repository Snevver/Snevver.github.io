Pong

Over dit project

Dit is een versie van pong die ik heb gemaakt met behulp van Phaser. Dat is een C# framework waarmee je games voor in de browser kan maken. Aangezien ik interesse heb ik het ontwikkelen van games heb ik pong gemaakt om te oefenen. Het is niet precies zoals het originele pong, maar het is zeker een goed begin. Ik heb erg veel geleerd over hoe je C# gebruikt om vershillende dingen te maken. Ik wil hier zeker nog heel veel mee doen. Dit project staat helaas niet op Github, dus is de media het enige wat je nu van het spel kan zien...

Gebruikte technologieën

  • - Phaser
